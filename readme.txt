== Installation ==

= Minimum Requirements =

* WordPress 5.2 or greater
* WooCommerce 3.0.0 or greater
* PHP version 7.2 or greater
* MySQL version 5.6 or greater

= Automatic Installation =

1. Go to WordPress Admin > Plugins > Add New
2. Search for "ShopMetrics for WooCommerce"
3. Click "Install Now" and then "Activate"
4. Follow the onboarding wizard to complete setup

= Manual Installation =

1. Download the plugin zip file
2. Go to WordPress Admin > Plugins > Add New > Upload Plugin
3. Choose the downloaded zip file and click "Install Now"
4. Activate the plugin
5. Follow the onboarding wizard to complete setup

= After Installation =

1. Navigate to **ShopMetrics** in your WordPress admin menu
2. Complete the quick onboarding process
3. Configure your Cost of Goods Sold settings (optional)
4. Start exploring your analytics dashboard!

== Frequently Asked Questions ==

= Is WooCommerce required? =

Yes, this plugin requires WooCommerce to be installed and activated. It's specifically designed to work with WooCommerce stores.

= Do I need a ShopMetrics account? =

The plugin automatically creates a secure connection to our analytics cloud during setup. No separate account registration is required.

= Will this slow down my website? =

No! The plugin is designed to be lightweight and uses background processes for data synchronization. Your store's performance won't be affected.

= Can I export my data? =

Yes, you can export your analytics data in various formats directly from the dashboard.

= Is my data secure? =

Absolutely! We use enterprise-grade encryption and follow strict security protocols to protect your data. We are fully GDPR compliant.

= What happens if I deactivate the plugin? =

Your data remains safely stored in our cloud for 30 days. If you reactivate the plugin during that time, all your historical data will be available again.

= Do you offer support? =

Yes! We provide comprehensive support through our documentation and support team.

= Can I customize the dashboard? =

The dashboard is designed to be comprehensive out of the box. We're continuously adding new features based on user feedback.

= Does this work with other WordPress ecommerce plugins? =

Currently, ShopMetrics is designed specifically for WooCommerce. Support for other ecommerce platforms may be added in future versions.

= What currencies are supported? =

ShopMetrics supports all currencies that WooCommerce supports. Multi-currency stores are fully supported.
